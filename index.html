SlideVise: Visual Preference Learning for
Code-Based Slide Generation

Abstract

Automated slide generation systems typically emphasize semantic correctness while failing to account for the inherently visual nature of slides, which often results in issues such as element overlap, text occlusion, and disorganized layouts. To address these limitations, we present a visually enhanced training strategy that enables visual quality signals to directly affect component selection, positioning, and sizing during the generation process. The proposed method consists of two stages. First, we build an automated data synthesis pipeline that integrates visual evaluation to construct a training corpus. This corpus encodes essential visual priors, including component positioning, sizing, and spatial organization. Supervised fine-tuning on this dataset establishes a reliable mapping from user requirements to compilable code. Second, we synthesize paired samples via parameter perturbation and use the visual differences between them as reward signals to further optimize the model, thereby effectively injecting continuous and differentiable visual supervision into the generative process. Unlike conventional generate-then-feedback paradigms, the proposed method tightly couples visual guidance with code synthesis, enabling the model to proactively avoid visually suboptimal configurations. Experimental results demonstrate that our proposed approach reduces failures in layout while preserving semantic fidelity, leading to clearer structures and more consistent visual organization. Furthermore, scores based on VLMs align well with human judgments under identical criteria. The code and model weights will be made publicly available.

How does it work?


SlideVise employs a two-stage training strategy. Initially, it constructs a high-quality dataset incorporating visual evaluation via an automated pipeline, leveraging SFT to establish a reliable mapping from user requirements to compilable code. Subsequently, visual reinforcement training is conducted, introducing Layout-Aware Slide Similarity (LASS) to quantify visual discrepancies and employing Weighted DPO to further reinforce the model's preference for high-quality layouts. Ultimately, the model ensures semantic correctness while proactively avoiding visual defects, generating slides that are both structurally coherent and aesthetically pleasing.
